Supercharged Linux SysAdmin One Liners

System Monitoring & Performance
• htop || top -c — real time system processes.
• ps aux --sort=-%mem \| head -10 — top memory hogs.
• ps aux --sort=-%cpu \| head -10 — top CPU users.
• du -sh */ \| sort -hr — directory disk usage sorted.
• find . -type f -exec ls -lh {} + \| sort -k5 -hr \| head -10 — largest files here.
• iotop -ao || iostat -x 1 — I/O intensive processes.
• ss -tulpn \| grep LISTEN — listening ports and services.
• iftop || nethogs — real time network traffic per process.
• uptime && free -h && df -h — quick health snapshot.
• grep "Failed password" /var/log/auth.log \| tail -10 — recent failed logins.
• dmesg -T | sed -e '…' — color enhanced kernel logs.
• watch -n 1 'ss -tuln' — monitor open ports live.
• ncdu — interactive disk usage analyzer.
• pv access.log \| gzip > access.log.gz — monitor compression progress.
• systemctl --type=service --state=running | awk '{print $1}' | xargs -I {} sh -c 'echo -n "{}: "; systemctl show {} --property=MemoryCurrent --value' — memory usage by service (via cgroups) 
• ps -eo pid,ppid,cmd,%mem,%cpu --sort=-%cpu \| head -10 — find processes causing high load.
• ss -s \| grep -E "estab|listen|time-wait" — network connections by state.
• iotop -b -n 1 \| head -20 — disk I/O by process (batch mode).
• for file in /proc/*/status; do awk '/VmSwap|Name/{printf $2 " " $3}END{print ""}' $file; done \| sort -k 2 -n \| tail -10 — swap usage by process.

File Operations & Management
• find /path -type f -mtime +30 -delete — purge old files.
• find / -type f -size +100M 2>/dev/null — grab oversize files.
• tar czf backup_$(date +%Y%m%d_%H%M%S).tar.gz /path/to/backup — timestamped backup.
• rsync -avh --progress source/ destination/ — copy with progress.
• fdupes -r /path — duplicate file finder.
• find /path -type f -exec md5sum {} + \| sort \| uniq -w32 -dD — dedupe via hash.
• rename 's/ /_/g' * — strip spaces in filenames.
• grep -r "search_text" /path --include="*.conf" — search within conf files.
• diff -u file1 file2 \| colordiff — colored diff display.
• mkdir -p /path && touch /path/file.txt — nested path + file.
• extract() { … } — multipurpose archive extractor.
• mount \| column -t — formatted mount output.
• less +F logfile — less as tail -f.
• find /path -type f -mmin -30 — find files modified in the last N minutes.
• rsync -av --delete source/ destination/ — sync directories with deletion.
• find /path -type l ! -exec test -e {} \; -print — find broken symlinks.
• find . -type f | sed 's/.*\.//' | sort | uniq -c | sort -nr — count files by extension.
• find /path -type f -perm 0777 -ls — find files with 0777 permissions.
